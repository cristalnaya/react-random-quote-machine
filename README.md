# React Build a Random Quote Machine

freeCodeCamp Random Quote Machine is a simple app, built with react.

## Install and Run the app

* Clone this repo in your terminal with the command `git clone https://github.com/cristalnaya/react-random-quote-machine.git`
* or just download it to your computer
* change directory into `cd react-random-quote-machine`
* run `npm install` in your terminal
* run `npm start` and the app will run locally on port: 3000

Note: [Node JS](https://nodejs.org/en/download/) and [NPM (Node Package Manager)](https://www.npmjs.com/get-npm) is required to install and run the application.
